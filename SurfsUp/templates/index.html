Welcome to the Hawaii Climate App API

Available Routes:

Route Description
/api/v1.0/precipitation Precipitation in the last 12 months
/api/v1.0/stations List of all Hawaii stations
/api/v1.0/tobs Temperature Observations of the most active station for a year
/api/v1.0/<start> Min, Average and Max Temperature starting from a specified date(YYYYMMDD)
/api/v1.0/<start>/<end> Min,Average and Max Temperature between a start and end date(YYYYMMDD). Input End date must be greater than Start date
